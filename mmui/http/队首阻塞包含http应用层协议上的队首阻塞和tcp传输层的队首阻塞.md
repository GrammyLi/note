http2.0 提出了 frame 和 stream 的概念， 把请求数据分成了不同的二进制 frame，有 strameid 标识，包装进不同的 stream 通过同一个 tcp 管道进行并发的多路复用传输， 接收端根据 streamid 来进行组装，解决了队首阻塞的问题。

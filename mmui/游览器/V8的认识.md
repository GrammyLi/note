核心能力：对虚拟机的理解。这一部分内容能考查出对操作系统、编译、虚拟机相关的理解。
一、V8 的主要功能

执行 JS 代码、管理内存分配、内存回收（垃圾回收）、优化代码执行。

二、执行 JS 代码

执行 JavaScript 代码：
脚本 -> 词法分析 -> 语法分析 -> AST -> 字节码 -> 解释执行（可以简单理解为字节码边解释边执行）。


在解释执行的过程中运用了 JIT 实时优化（热点代码直接变异成机器码）。

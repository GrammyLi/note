添加到暂存区
```git
git add .
 ```

添加描述
```git
git commit -m 'fix: 描述的内容'
 ```

拉取代码
```git
git pull origin 分支名称
 ```

推送代码
```git
git push
 ```
 
提交本地缓存

```git
git stash save "描述内容"
 ```
  
内容
```git
gst
 ```
 
提交的 list
```git
git stash list
 ```

pop
```git
git stash pop stash@{0}
 ```
